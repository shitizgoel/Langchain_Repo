# LangChain

This repository contains notes from the LangChain course by Ansh Lamba on YouTube.

## Structure

- `CH-1/` - Langchain Basics
- `CH-2/` - Chains
- `CH-3/` - React Agent in Langchain

## Usage

1. Open the course material folders.
2. Run example scripts using a Python environment.
3. Add new notes or code to the appropriate folder.

## Requirements

- Python 3.10 or newer
- Install langchain

## Initial steps
- uv init : Initialization
- uv sync : Make virtual environment
- source .venv/bin/activate : Activate environemt
- uv add langchain : To add any package (Make sure venv is activated)
- uv add -r requirements.txt
