# LangChain

This repository contains notes from the LangChain course by Ansh Lamba on YouTube.

## Structure

- `CH-1/` - Langchain Basics
- `CH-2/` - Chains
- `CH-3/` - React Agent in Langchain

## Usage

1. Open the course material folders.
2. Run example scripts using a Python environment.
3. Add new notes or code to the appropriate folder.

## Requirements

- Python 3.10 or newer
- Install langchain
